# Fonts

This directory contains self-hosted font files for the portfolio.

## Required Files

Download the following fonts and place them in this directory:

### Geist Variable (Sans)
- **File**: `GeistVF.woff2`
- **Source**: https://github.com/vercel/geist-font/releases
- **License**: OFL (SIL Open Font License)
- **Weight range**: 100-900 (variable)

### Geist Mono Variable
- **File**: `GeistMonoVF.woff2`
- **Source**: https://github.com/vercel/geist-font/releases
- **License**: OFL (SIL Open Font License)
- **Weight range**: 100-900 (variable)

## Installation Steps

1. Visit https://github.com/vercel/geist-font/releases
2. Download the latest release
3. Extract the archive
4. Copy `GeistVF.woff2` and `GeistMonoVF.woff2` to this directory
5. Restart the dev server if running
